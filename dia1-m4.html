Usando o Teclado

⏰ 14h50 – 17h · ~2h10 (com exercícios)

O teclado tem muitas teclas, mas não precisamos saber todas de uma vez. Vamos conhecer as mais importantes para digitar textos do dia a dia.

🔑 Grupos de teclas importantes

🔤 Letras e acentos
ABC...Z
26 letras + teclas de acento (´ ~ ^)
🔢 Números
1234567890
Na fila de cima ou no teclado numérico
✨ Teclas especiais
← BackspaceEnter ↵Espaço
Backspace = apagar. Enter = confirmar. Espaço = separar palavras.
🔠 Maiúscula
⇧ ShiftCaps Lock
Shift: segure + tecla = maiúscula (só uma).
Caps Lock: trava maiúsculas.

🇧🇷 Como digitar acentos

Atividades Práticas — Mão na Massa!
4 atividades · faça no seu próprio computador

📝
Bloco de Notas — Digitação
⌨️ Teclado⏱ ~15 minFácil
Como abrir: Iniciar → digitar "Bloco de Notas" → Enter
  1. Digite seu nome completo. Use Shift para a primeira letra maiúscula. Pressione Enter para pular linha.
  2. Digite seu endereço completo (rua, número, bairro, cidade).
  3. Digite o texto abaixo linha por linha:
    João Câmara é uma cidade do Rio Grande do Norte.
    O computador é uma ferramenta muito útil no dia a dia.
    Aprender informática não é difícil — é só prática!
    Hoje dei meu primeiro passo no mundo digital.
    Ação, coração, ônibus, mamãe, café — eu consigo!
  4. Use Ctrl + S para salvar. Nome do arquivo: "dia1_meu_nome".
✅O arquivo salvo com seu nome, endereço e o texto digitado.
2
🎨
Paint — Prática com o Mouse
🖱️ Mouse⏱ ~20 minMédio
Como abrir: Iniciar → digitar "Paint" → Enter
  1. Escolha o Lápis e desenhe uma casa simples: um quadrado com triângulo em cima.
  2. Use Formas para adicionar uma porta (retângulo) e o sol (oval).
  3. Use o Balde de tinta para colorir cada parte com cor diferente.
  4. Use a ferramenta Texto (A) para escrever seu nome no desenho.
  5. Salve com Ctrl + S como "minha_casa".
✅Desenho colorido com seu nome escrito.
3
🔢
Calculadora — Cliques de Precisão
🖱️ Mouse⏱ ~10 minFácil
Como abrir: Iniciar → digitar "Calculadora" → Enter
  1. Calcule: 25 + 37 = ?
  2. Calcule: 100 − 43 = ?
  3. Calcule: 12 × 5 = ?
  4. Calcule sua idade: 2026 menos seu ano de nascimento.
✅Os 4 resultados anotados no Bloco de Notas.
4
🖥️
Área de Trabalho — Janelas e Ícones
🖱️ Mouse⏱ ~10 minMédio
Objetivo: Aprender a controlar janelas
  1. Abra o Bloco de Notas e o Paint ao mesmo tempo.
  2. Clique em cada programa na barra de tarefas para alternar entre eles.
  3. Clique no botão — (minimizar). Clique na barra de tarefas para trazer de volta.
  4. Clique no botão □ (maximizar) para ocupar a tela toda.
  5. Arraste a barra de título para mover a janela. Coloque as duas lado a lado.
✅Os dois programas abertos e visíveis ao mesmo tempo.
