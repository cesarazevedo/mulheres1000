Momento 2 — O que é um Computador?

⏰ 13h30 – 14h  ·  30 minutos
Hardware CPU RAM HD Monitor

Um computador é uma máquina que recebe informações, processa e devolve um resultado. Para entender as partes, vamos usar analogias do nosso dia a dia.

🏠 Analogia: O Computador é como uma Casa

🧠
CPU / Processador
= Cérebro
Faz todos os cálculos e decisões. Quanto mais rápido, mais ágil o computador.
🪑
Memória RAM
= Mesa de Trabalho
Guarda o que você está usando agora. Desligar o PC limpa a mesa.
🗄️
HD / SSD
= Armário / Gaveta
Guarda tudo permanentemente: fotos, documentos, músicas. Fica salvo mesmo desligando.
🪟
Monitor
= Janela
Mostra o que está acontecendo dentro do computador. Não armazena nada.
🖱️
Mouse
= Mão Virtual
Você move e aponta com a mão na tela. Clica para escolher e selecionar.
⌨️
Teclado
= Voz Digital
Você digita para "falar" com o computador: escrever textos, senhas, endereços.

📊 Resumo comparativo

Parte Analogia O que faz Tipo
CPU 🧠 Cérebro Processa tudo Interno
RAM 🪑 Mesa Memória temporária Interno
HD / SSD 🗄️ Armário Armazenamento permanente Interno
Monitor 🪟 Janela Exibe a imagem Saída
Mouse 🖱️ Mão Controle e seleção Entrada
Teclado ✍️ Voz Digitar informações Entrada
💡 Dica fácil de lembrar: Entrada = você envia ao computador (teclado, mouse). Saída = o computador te devolve (monitor, som). Processamento = o computador pensa (CPU + RAM). Armazenamento = guarda (HD/SSD).
🖱️

Momento 3 — Ligando e Usando o Mouse

⏰ 14h – 14h30  ·  30 minutos
Mouse Windows Área de Trabalho Prática

⚡ Como ligar e desligar corretamente

1
Ligar o computador
Pressione o botão de energia (Power). Aguarde o Windows carregar — pode demorar alguns segundos. Não aperte o botão de novo!
2
Desligar pelo Windows
Clique em Iniciar (botão do Windows) → Desligar. Espere o computador se apagar completamente antes de sair. Nunca aperte o botão de energia para desligar durante o uso!
⚠️ Atenção: Desligar no "murro" (botão de energia) pode corromper arquivos e danificar o computador. Sempre use o menu Iniciar → Desligar.

🖱️ Conhecendo o Mouse

👆
Botão Esquerdo — o principal
1 clique: selecionar / abrir link
2 cliques rápidos: abrir pasta ou programa
🖱️
Botão Direito — o menu de opções
1 clique: abre um menu com opções (copiar, colar, renomear…)
🔄
Scroll (rodinha) — rolar a página
Role para cima ou para baixo para ver mais conteúdo sem mover o mouse.
✊
Arrastar e soltar
Clique, segure o botão esquerdo, arraste e solte. Usado para mover ícones.

🖥️ A Área de Trabalho (Desktop)

📌
Ícones
São as figuras na tela. Duplo-clique para abrir. Botão direito para opções.
📌
Barra de Tarefas (embaixo)
Mostra os programas abertos e o botão Iniciar (o símbolo do Windows). Clique no botão Iniciar para acessar programas.
📌
Janelas
Cada programa abre numa "janela". No canto superior direito: — minimiza, □ maximiza, ✕ fecha.
💡 Prática agora: Mova o cursor pelo desktop. Abra e feche uma pasta clicando duas vezes. Clique com o botão direito na área de trabalho e veja as opções que aparecem.
⌨️

Momento 4 — Usando o Teclado

⏰ 14h50 – 17h  ·  ~2h10 (com exercício)
Teclado Acentos Bloco de Notas Prática

O teclado tem muitas teclas, mas não precisamos saber todas de uma vez. Vamos conhecer as mais importantes para digitar textos do dia a dia.

🔑 Grupos de teclas importantes

🔤 Letras e acentos
ABC DEF G...Z
26 letras + teclas de acento (´ ` ~ ^)
🔢 Números
123 456 789 0
Na fila de cima ou no teclado numérico (se houver)
✨ Teclas especiais mais usadas
← Backspace Enter ↵ Espaço
Backspace = apagar. Enter = confirmar / pular linha. Espaço = separar palavras.
🔠 Maiúscula e minúscula
⇧ Shift Caps Lock
Shift: segure + tecla = letra maiúscula (só uma).
Caps Lock: liga/desliga maiúsculas constantes.

🇧🇷 Como digitar acentos

No teclado brasileiro, os acentos funcionam em duas etapas: você pressiona a tecla do acento, solta, depois pressiona a letra.

💡 Dica importante: Se você pressionar a tecla de acento e digitar espaço em vez de uma letra, aparece o símbolo sozinho: ´ ou ~. Isso é útil para escrever o símbolo sem acento.
⚠️ Erro comum: Se a tecla de acento não funcionar, verifique se o Caps Lock está ligado — às vezes isso interfere. Desligue e tente novamente.

✍️ Exercício Prático — Bloco de Notas

Vamos praticar tudo que aprendemos hoje! Abra o Bloco de Notas e siga os passos abaixo. Tire uma foto ou screenshot para mostrar que completou.

1
Abra o Bloco de Notas: Clique em Iniciar → escreva "Bloco de Notas" → pressione Enter para abrir.
2
Digite seu nome completo. Use Shift para a primeira letra maiúscula. Pressione Enter para pular de linha.
3
Digite seu endereço (rua, número, bairro, cidade). Tente escrever com acentos corretos: João Câmara, não, ação, água.
4
Escreva uma frase sobre o que você espera aprender neste curso. Pode ser com suas palavras, sem preocupar com erros.
5
Mostre para o professor e para a colega ao lado. 🎉 Parabéns — você acabou de usar o computador!
